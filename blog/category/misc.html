Build vim with python support via Homebrew

Wed 20 February 2013

vim에서 Python을 보다 편하기 쓰기 위해 https://github.com/klen/python-mode 를 설치하려다 보니, 사용중인 vim에 python 지원이 없음을 알게 되었다.

당연하지만, OS X 에 기본 설치되어 있는 vim은 language support가 빠진채 빌드되어 있다.

Homebrew에 포함된 vim.rb formula는 언어 지원을 위한 옵션을 포함하고 있는데, 아래 명령으로 지원하는 옵션을 확인할 수 있다.

$ brew options vim

Python 지원을 포함하려면 아래와 같이 설치한다.

$ brew install vim --with-python

vim.rb의 25-31L를 보면, --with- 옵션은 configure 시에 --enable-interp 옵션으로 변경되어 들어간다.
